JavaScript

JavaScript de maneira análoga, é uma linguagem de programação que sua o navegador para funcionar, diferente do HTML que é uma linguagem de marcação.
O JavaScript pode fazer coisas como alterar a cor de algo ao pressionar um botão. De forma simples, é possivel tornar usar o js para manipular tanto HTML quanto o CSS de um elemento ou de uma página inteira.
Como por exemplo, alterar a cor de um título para outra cor, ou alterar oque está escrito nele.

Para inserirmos um função ou um Script para esse botão, podemos linkar um arquivo .js ou adicionar a tag < Script > no (fim da página de preferência).

A estrutura do JS depende do que esta Escrevendo. De forma explicativa:

  • var

    Var é a abreviação de Variável, e com essa tag, podemos armazenar informação, como o número de algo, ou uma palavra de uma oração.
    Fazemos isso Escrevendo: Var NomeDaVariável = função.

  • Function

    Function é quando usamos quando queremos ter algo que faça alguma função específica. No Exemplo anterior, usamo o Fucntion e atribuimos o nome de MyFunction, e nisso colocamos um paretese (), onde podemos atribuir um valor aquela função, e em seguida, abrindo chaves para inserir o códigos que irá designar o funcionamento daquela Function.

  • Escrevendo uma função

    Para escrever uma função, começamos com document: Document é a porta de entrada para todo o código, ja que toda pagina, carrega um document como objeto.
    Em seguida, usamos um método para ele ter acesso ao que queremos que seja feito, há muitas tags que o fazem, como por exemplo, para referênciarmos qual elemento no HTML queremos mudar, foi usado o getElementById que retorna o valor ao ID atribuido. Em seguida, usamo o método de entrada, que nessa caso foi o innerHTML, que indica um tipo de alteração ao HTML, e em seguida, um = com o valor que queremos atribuir ao elemento, que nessa caso, por ser uma string, colocamos entre aspas.
