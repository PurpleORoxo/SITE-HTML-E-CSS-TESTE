Tags para Links, imagens, audios, videos e etc

Inicialmente, usando a tag < img > é possivel anexar uma imagem, onde dentro da tag, precisamos inserir src="" para definir o caminho para chegar a imagem, seja na web, seja na maquina local. A tag alt="", usamos para inserir um título quando pessoas não conseguem carregar a imagem, mas pelo menos possuem a descrição da mesma.

Usamos a tag < Video > para anexar um video, e dentro do mesmo, adicionamos Control para poder executalo. Dentro da tag, adicionamos a tag < source > e dentro dela o src="" para fazer a referencia ao caminho do video, apenas de um diretório da maquina e type para especificar o tipo de video, ou seja, a extensão.

Usamos a tag < Audio >, para adicionar um arquivo de audio, e igualmente a Video, adicionamos dentro dela o Control para poder executar e adicionamos dentr da tag Source e dentro desta adicionamos o src="" para referenciar o audio e type="" o tipo de extensão.

Para anexa um Link usamos a tag < a >, nela adicionamos a tag href="" que será para referenciar o link de uma outra página. Também podemos anexar uma imagem e usa-la como botão para abrir o link. Usando a tag Target="_blank" podemos fazer com que o link abra em outra aba automaticamente.

Para Links de Videos, usamos a tag < frame >, a qual anexas um card do Youtube para executar, porem diferente da Tag Video, não é necessario adicionar control e nem dentro dela a tag source, mas adicionar src="" para anexas o link do video. Porém, o link esta determinado como watch, então retiramos a parte "watch?v=" e adicionamos embed/, então o link ficaria https://www.youtube.com/embed/AexeWtlbEBk

Tags para Botões e box para inserir Textos

Para criar botões, usasse a tag < Button >, que irá gerar uma caixa de texto clicavel. Caso queira adicionar algum Texto, é necessario adicionar um Type="" para especificar, e caso queira que algo ocorra ao clicar, usamos o onclick="", para realizar algo quand esse botão for apertado.


O exemplo a baixo mostra apenas o botão propriamente dito. Porém, se clicarmos nele, nada ocorre.

Para que algo ocorra, é necessario adicionar onclick="", pois isso especificará o que deve ocorrer ao clicar o botão. A vários tipos de elementos que podem ser o utilizados dentro do onclick="", como alert, que irá gerar uma caixa de texto que falará algo, após o botão ser clicado.

Tag de Criação de Tabela

Para criação de uma tabela, usamos a tag < Table >, que irá gerar uma estrutura, porém, para a criação das colunas e linhas usamos < tr > para a criação da Linha, e < td > para a criação da coluna. Para alteração do Estilo da tabela, inserimos suas configurações dentro da tag < Table antes do >. Usando colspan dentro da < td > , para escolher o tamanho da linha, equivalente a um certo tamanho de coluna. Igualmente, é possivel escolher o tamanha da coluna, usando rowspan
